# RoboCamCal(Robot Camera Calibration)
This project implements some calibrations including hand-eye calibration, camera calibration, stereo camera calibration;

## Dependence
1. OpenCV(>=3.0.0)
2. Visp
3. gflags

## How to build
```bash
cd PROJECT_DIR
mkdir build
cd build
cmake ..
make
```
